Ansible Provisioning

The underlying nodes and cluster configuration were provisioned using Ansible. Below are key excerpts from the playbooks.

Playbook: Initial Node Setup 
---
- name: Configure base server
  hosts: all
  become: yes
  tasks:
    - name: Update and upgrade apt packages
      apt:
        update_cache: yes
        upgrade: dist
...
Playbook: K3s Cluster Installation 
---
- name: Install K3s cluster
  hosts: master
  become: yes
  tasks:
    - name: Install K3s from script
      shell: "curl -sfL https://get.k3s.io | sh -"
      args:
        creates: /usr/local/bin/k3s
...
